How to Connect

1. First of all, you must create an account. The account is used to log into both the game and our website. Click here to open the registration page

2. Download and install the World of Warcraft 1.12.1 (5875) client. You can download it from Internet Archive (look for WoW-1.12.1_install.rar).

3. Open your World of Warcraft installation folder and find the file realmlist.wtf (located inside the root directory or Data/enUS / Data/enGB depending on your client).

4. Open realmlist.wtf with a text editor, erase all text, and set it to:

set realmlist logon.example.com
